# Full-Stack Task Management Workspace

A robust, lightweight task workspace and CRUD microservice engineered using Node.js and Express.js. This service provides structured endpoint paths for managing day-to-day organizational tasks, utilizing synchronized JSON schema local file storage with optimized operational tracking.

## Technical Stack
* **Runtime Environment:** Node.js (v18+)
* **Backend Framework:** Express.js
* **Persistence Layer:** Structured JSON native file streaming core
* **Unique ID Generation:** Cryptographic random hashing utility

## Key Features
* Asynchronous REST API routing managing complete item data lifecycles (CRUD)
* Automated time-stamped task compilation sorting records dynamically
* Direct operational toggle points managing binary state adjustments
* 100% responsive user management console designed with modern CSS3 layout paradigms

## Installation and Setup
1. Clone the repository to your environment
2. Execute dependency configuration: `npm install`
3. Launch local execution cycle: `npm start`
4. Access the workspace view via: `http://localhost:5000`
